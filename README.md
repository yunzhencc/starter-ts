## ⌨️ 本地开发

请使用 [pnpm](https://pnpm.io/zh) 作为包管理工具

```sh
# 克隆项目
git clone git@github.com:yunzhencc/starter-ts.git

# 进入项目目录
cd starter-ts

# 安装依赖
pnpm install
```
